Категория: Покупки / Интернет-магазин

Краткое описание:
Matule – удобное приложение для онлайн-покупок. Ищете товары, добавляйте в избранное, оформляйте заказы и следите за историей – всё в одном месте.

Полное описание:
Matule – это современное приложение для покупок, которое делает процесс выбора и заказа товаров простым и приятным.

Основные функции:
- Быстрая авторизация и регистрация (вход по email/паролю, восстановление пароля с OTP-кодом).
- Удобный каталог товаров с карточками (изображения, название, цена).
- Возможность добавлять товары в избранное и корзину.
- Детальная карточка товара: описание, переключение свайпом, кнопка "Подробнее" для разворачивания описания.
- Поиск товаров с сохранением истории поиска (без учёта регистра).
- Корзина: изменение количества товаров, удаление свайпом, итоговая сумма.
- Оформление заказа с редактируемыми полями (телефон, email, адрес) и автозаполнением адреса по геопозиции.
- История заказов с группировкой по датам (сегодня, вчера, другие даты), повтор заказа (свайп вправо) и отмена (свайп влево).
- Профиль пользователя: редактирование личных данных, смена аватарки (камера, галерея).
- Карта лояльности со штрих-кодом, генерируемым из ID пользователя (открывается на весь экран с автоматическим повышением яркости).
- Уведомления о важных событиях (список, получаемые в реальном времени).
- Боковое меню для быстрого доступа к профилю, истории заказов, избранному, уведомлениям и выходу из аккаунта.
- Тёмная и светлая тема (подстраивается под систему).

Приложение разработано с использованием чистой архитектуры, что гарантирует стабильность и лёгкость поддержки. Все медиаресурсы кэшируются для быстрой работы без интернета.
